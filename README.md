# trailine <진행중>

등산 및 트레킹 코스에 대한 가이드를 알려주는 서비스

[프로젝트 진행 현황](https://github.com/orgs/sweetcase-production/projects/8)
* [회원가입 관련 API 구현](https://github.com/sweetcase-production/trailine/pull/13)


## 개발 목표

* 코스 길이, 난이도, 일주일 날시, 주의사항 등 최대한 상세하게 알려줄 수 있게 개발
* GPS 기반으로 코스 인증이 된 유저에 한해 코스 이용 후가 작성
* 코스 여러개를 모아 원하는 코스집을 만들 수 있음


## 사용 기술

|포지션|사용기술|
|---|---|
|Backend|Python3.13, Django/DRF, Redis, PostgreSQL|
|Frontend(WEB)|Typescript, React|
